# Active

Sometimes you may need to silence Estimtest. For example, you may only need the element active when experimenting. Setting `active` to `false` or `"false"` will hide the Estimtest menus.

:::caution
This will not remove the element declaration from the source code; therefore, this does not decrease the payload size. If you need to remove the component entirely you may try looking at the solutions presented in [the examples](https://github.com/KyleSmith0905/estimtest/tree/main/examples).
:::
